fuz_ui is a Svelte UI library with components and helpers for making zippy websites. It builds on fuz_css, which is based on CSS custom properties, and fuz_ui continues those patterns. It's in early alpha and will have many breaking changes.

Compared to most UI libraries, fuz_ui has fewer features and more opinions, but the eventual goal is to be fully-featured with high flexibility. Fuz emphasizes capability and efficiency and tries to be simple for those goals.

To learn more see the docs and readme. Feel free to take the ideas and code for your own purposes.
